### Description

```
    Реализация STUB сервера со следующими методами-заглушами:
    -> получение списка пользователей
    -> получение списка курсов
    -> получение списка оценок
```

### Install

```
    git clone https://github.com/justiwannauseit/home_task_4 
```

### Run

```
    В проекте используется два модуля:
    -> Сервис получения пользователей
    -> Тестовый фреймворк
    .................................................................
    Для корректной работы тестового фреймворка необходимо поднять
    Stub сервер. Затем запускать тесты.
    (!) Примечание: STUB сервер поднимается на http://localhost:8080/
```
